# Underthesea Bank Sentiment

Bộ dữ liệu tổng hợp các bài viết trên mạng xã hội về lĩnh vực ngân hàng của underthesea

### Thống kê 

Tổng số 2471 bài viết (train: 1977 bài viết, test: 494 bài viết)

14 nhãn khía cạnh 

```
- ACCOUNT
- CARD
- CUSTOMER_SUPPORT
- DISCOUNT
- INTEREST_RATE
- INTERNET_BANKING
- LOAN
- MONEY_TRANSFER
- OTHER
- PAYMENT
- PROMOTION
- SAVING
- SECURITY
- TRADEMARK
```

3 nhãn cảm xúc

```
- positive
- neutral
- negative 
```
